Java 21 software designed to create raster images for use as templates for CNC milling machines when cutting out various geometric shapes. The program is written in Java 21. If JRE 21 is installed on the system, you can use Shapixer.jar to run the program. To use the program under Windows OS without JRE installed, download ( https://cutt.ly/St7BSGYS ) and install the ShapixerSetup.exe installer. Project website: https://sites.google.com/view/autester
